# Runbook

## Root app commands

- `npm run dev`: start Expo development server.
- `npm run android`: run Expo app for Android.
- `npm run ios`: run Expo app for iOS.
- `npm run web`: run Expo app for web.
- `npm run build:web`: export web build and generate service worker.
- `npm run lint`: lint the root app.

## Backend commands

- Run commands from `backend/`.
- `npm run dev`: start backend in watch mode.
- `npm run build`: bundle backend.
- `npm run typecheck`: TypeScript checking.
- `npm run start`: start built backend.
- `npm run db:push`: update schema and run migrations.

## Working habits

- Update `docs/project_ledger.md` at the start and end of meaningful work.
- Add durable decisions to `docs/06-decisions/`.
- Use the repo maps before making broad structural changes.
